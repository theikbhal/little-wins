# Little Wins 🎉

A tiny app for ADHD brains. Log small wins, celebrate them, and build momentum.

- **iPhone App** — React Native / Expo (in development)
- **Mac App** — coming soon
- **Android App** — coming soon
- **Chrome Extension** — coming soon
- **Landing Page** — deployed on Vercel

## Stack

- [Expo](https://expo.dev) (React Native)
- AsyncStorage (local only — no cloud)
- TypeScript

## Project Structure

```
apps/
  little-wins/
    iphone-app/       # Expo React Native app
    mac-app/          # (future)
    android-app/      # (future)
    chrome-extension/ # (future)
    landing-page/     # Vercel landing page
```

## Development

```bash
cd iphone-app
npm install
npx expo start --ios
```

## Philosophy

> "We overestimate what we can do in a day and underestimate what we can do in a year."
> — The power of small wins compounds.

Privacy-first. No accounts. No cloud. Just you and your wins.
